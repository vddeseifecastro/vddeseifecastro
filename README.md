## Projects

**[shopeasy](https://github.com/vddeseifecastro/shopeasy)** — Full stack e-commerce app with JWT auth, product catalog, shopping cart, order management and admin panel.

**[bizintel-dashboard](https://github.com/vddeseifecastro/bizintel-dashboard)** — AI-powered business intelligence dashboard. Upload CSV data, visualize sales metrics and get AI insights using local or cloud LLMs.

**[Horarios-Ingenieria-2025](https://github.com/vddeseifecastro/Horarios-Ingenieria-2025)** — Academic schedule management system with drag-and-drop builder, multi-user support and PDF/Excel export.

**[biblioteca-final](https://github.com/vddeseifecastro/biblioteca-final)** — Full stack library management system built with Go, Gin and GORM. JWT auth, book catalog and loan management.

**[rag-enterprise-chatbot](https://github.com/vddeseifecastro/rag-enterprise-chatbot)** — Document Q&A system using RAG, FAISS and OpenRouter. Chat with PDFs, DOCX and TXT files in any language. 100% free.

**[banco_turnos](https://github.com/vddeseifecastro/banco_turnos)** — Bank queue management system with customer self-service web, public display screen and desktop admin panel.

**[tasknet-ai](https://github.com/vddeseifecastro/tasknet-ai)** — AI-powered Kanban board with drag & drop tasks, priority suggestions, subtasks and time estimates powered by local or cloud LLMs.

**[pricehawk](https://github.com/vddeseifecastro/pricehawk)** — Real-time price monitor with web scraping and WebSockets. Track products from any website and get instant alerts when prices drop.

**[moneymate](https://github.com/vddeseifecastro/moneymate)** — Personal finance PWA built with React. Track expenses, manage budgets and visualize spending trends.

**[workflow-task-manager](https://github.com/vddeseifecastro/workflow-task-manager)** — Full stack task manager with FastAPI, JWT auth, SQLite and Vanilla JS.

**[pokedex](https://github.com/vddeseifecastro/pokedex)** — Full Pokédex with 1025 Pokémon, special forms, comparator and real-time filters — Vanilla JS + PokéAPI.

**[sudoku-x](https://github.com/vddeseifecastro/sudoku-x)** — Scientific Sudoku with AI solver. 4 difficulty levels and step-by-step logical reasoning. Single HTML file — open and play instantly.
